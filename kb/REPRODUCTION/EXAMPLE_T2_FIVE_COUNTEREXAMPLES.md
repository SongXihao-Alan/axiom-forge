# 任务 T2 示例:5 个 SC 违反案例

> 任务 T2 的"参考输出"——展示**什么样的反例算合格**。
> 这是组织者提供的范例, 不是"标准答案"。

---

## 反例 1:非线性 f(quadratic)

**设置**:
- 真实结构: `f(X) = X_1^2 + 0 * X_2`, 即 X_1 平方项是唯一真实相关
- 预测器: `f̂(X) = linear model, 训练后得到 f̂(X) = 0.5 * X_1 + 0.3 * X_2` (X_1 系数 0.5, X_2 系数 0.3)
- **关键**: f̂ 学到了 X_2 的 spurious 关系

**SI 计算**:
- SI_1(f) = E_X[|2 X_1|] = 2 * E[|X_1|] > 0 (假设 X_1 期望不为 0)
- SI_2(f) = 0 (因为 f 与 X_2 无关)

**SHAP 归因**(以 f̂ 为基础):
- φ_1 > 0 (因为 f̂ 系数 0.5)
- φ_2 > 0 (因为 f̂ 系数 0.3,这是 spurious)

**SC 检验**:
- SI_2(f) = 0, SC 不约束 ✓
- SI_1(f) > 0, SC 要求 ∃x: φ_1 > 0 ✓ (满足)

**结论**:**这个反例不违反 SC**!但违反"无 spurious attribution"原则。

**锚解释**:
- **经验锚**: 跨文化共识——"归因不应反映预测器偏见"
- **哲学锚**: 现象学——"回到 f 本身, 不是 f̂ 学到的"

## 反例 2:mediator(mediator relationship)

**设置**:
- 真实结构: `X_1 → M → Y`, 其中 M = X_1 * 2, Y = M + ε
- 即 `f(X_1, M) = M = 2 * X_1` (X_1 通过 M 影响 Y)
- 预测器: `f̂(X) = 1.5 * X_1 + 0.5 * M` (X_1 直接系数 + M 直接系数)

**SI 计算**:
- SI_1(f) = |∂f/∂X_1| = |2 * 0| = 0? 这里用偏导不对, 因为 M 不是 X_1 的函数
- 重新理解: f 的定义域是 (X_1, M), f 与 X_1 无关, 所以 SI_1 = 0
- SI_M(f) = 1

**SHAP 归因**(在 (X_1, M) 特征空间):
- v(S) = E[f̂(X) | X_S] = E[1.5 X_1 + 0.5 M | X_S]
- 如果 S = ∅, v = E[1.5 X_1 + 0.5 M] = 1.5 μ_X1 + 0.5 μ_M
- 如果 S = {X_1}, v = 1.5 x_1 + 0.5 μ_M
- 如果 S = {M}, v = 1.5 μ_X1 + 0.5 m
- 如果 S = {X_1, M}, v = 1.5 x_1 + 0.5 m
- 边际贡献: φ_X1 = (1.5 x_1 + 0.5 μ_M) - (1.5 μ_X1 + 0.5 μ_M) = 1.5 (x_1 - μ_X1)
- φ_M = (1.5 μ_X1 + 0.5 m) - (1.5 μ_X1 + 0.5 μ_M) = 0.5 (m - μ_M)
- **φ_X1 和 φ_M 都 > 0**

**SC 检验**:
- SI_X1 = 0, SC 不约束
- SI_M = 1, SC 要求 φ_M > 0 ✓
- **SC 满足**

**关键洞察**:SC 在 mediator 场景下**不**自动违反——因为 SC 只看 f 的结构重要性,而 f 在 (X_1, M) 空间里与 X_1 无关。

**锚解释**:
- **经验锚**: 跨文化共识——"归因不应混淆直接和间接原因"
- **哲学锚**: 现象学——"f 的结构是 (X_1, M)→M, 不是 X_1 → Y"

## 反例 3:f̂ 完全丢弃真变量(场景 B 严格版)

**设置**:
- 真实结构: `f(X) = 10 * X_1` (X_1 系数 10, 强相关)
- 预测器: `f̂(X) = 0.5 * X_2` (完全忽略 X_1, 用 X_2 做预测)

**SI 计算**:
- SI_1(f) = 10 > 0
- SI_2(f) = 0

**SHAP 归因**:
- v(S) = E[f̂(X) | X_S] = E[0.5 X_2 | X_S]
- 如果 X_1 ⊥ X_2, 则 v(∅) = 0.5 μ_X2, v({X_1}) = 0.5 μ_X2 (因为 X_1 不影响 X_2 的条件分布)
- v({X_2}) = 0.5 x_2, v({X_1, X_2}) = 0.5 x_2
- 边际贡献: φ_X1 = 0 - 0 = 0, φ_X2 = 0.5 (x_2 - μ_X2)

**SC 检验**:
- SI_1 = 10 > 0, SC 要求 ∃x: φ_X1 > 0, **但 φ_X1 = 0 对所有 x** ✗ **SC 违反!**

**这是 SC 的核心反例**——**Impossibility 5.1 证明里的场景 B**。

**锚解释**:
- **经验锚**: 跨文化共识——"X_1 重要时归因应反映"
- **哲学锚**: 现象学——"f 的结构 = X_1 是关键, 归因应反映"

## 反例 4:spurious correlation(混淆变量)

**设置**:
- 真实结构: `f(X_1, X_2) = 3 * X_1` (X_1 是因, X_2 与 f 无关)
- 数据生成: `X_1 ↔ X_2` (P(X_1, X_2) 有相关性, eg `X_2 = X_1 + noise`)
- 预测器: `f̂(X) = 2 * X_1 + 1 * X_2` (学到 X_2 的 spurious 关联)

**SI 计算**:
- SI_1(f) = 3 > 0
- SI_2(f) = 0

**SHAP 归因**(以 f̂ 为基础, conditional):
- v(S) = E[f̂(X) | X_S] = E[2 X_1 + X_2 | X_S]
- 当 X_1 ⊥ X_2 假设下, v({X_1}) = 2 x_1 + μ_X2, v({X_2}) = 2 μ_X1 + x_2
- 边际贡献: φ_X1 > 0, φ_X2 > 0 (因为 X_2 在 f̂ 中有系数 1)

**SC 检验**:
- SI_1 > 0, φ_X1 > 0 ✓
- SI_2 = 0, SC 不约束 ✓
- **SC 满足**(但归因反映了 spurious 关系——这是 K20 批评的)

**关键洞察**:SC **不能**捕捉 spurious 关系——这是 SC 的**局限**。要捕捉这个需要 **Action-ability** 或因果图(cf. [H20] Causal Shapley)。

**锚解释**:
- **经验锚**: 跨文化共识——"X_2 在 f 里无关, 归因应反映"
- **哲学锚**: 现象学——"f 是 X_1 决定, 不是 X_2"

## 反例 5:f̂ 反映 mediator 但不反映 confounder

**设置**:
- 真实结构: `X_1 ← Z → X_2 → Y`, 其中 Z 是 confounder
- `f(X_1, X_2, Z) = X_2` (X_2 是直接因, X_1 和 Z 通过 X_2 间接影响)
- 预测器: `f̂(X) = 0.5 * X_2 + 0.3 * Z` (学到了 Z 的 spurious 直接效应)

**SI 计算**:
- SI_X1 = 0 (f 不直接含 X_1)
- SI_X2 = 1
- SI_Z = 0 (f 不直接含 Z)

**SHAP 归因**(以 f̂ 为基础):
- φ_X2 > 0 (来自 f̂ 系数 0.5)
- φ_Z > 0 (来自 f̂ 系数 0.3, spurious)
- φ_X1 ≈ 0

**SC 检验**:
- SI_X1 = 0, SC 不约束 ✓
- SI_X2 > 0, φ_X2 > 0 ✓
- SI_Z = 0, SC 不约束 ✓
- **SC 满足**(但归因反映 confounder 的 spurious 效应)

**锚解释**:
- **经验锚**: 跨文化共识——"Z 在 f 里无关"
- **哲学锚**: 现象学——"f 是 X_2 直接决定, Z 是 confounder"

## 5 个反例的总结

| # | 类型 | SC 违反? | 反 SC 违反的核心点 |
|---|---|---|---|
| 1 | quadratic + linear predictor | ✗ (不违反) | SHAP 反映 spurious,但不直接违反 SC |
| 2 | mediator | ✗ | 在 (X_1, M) 空间, f 不含 X_1,SC 不约束 |
| 3 | 场景 B 严格版(f̂ 丢真变量)| **✓ 违反** | SI_1 = 10 但 φ_X1 = 0 |
| 4 | spurious correlation | ✗ | 归因反映 spurious, 但 SC 不捕捉这个 |
| 5 | confounder | ✗ | 同 4,SC 不捕捉 spurious |

**关键发现**:**SC 的反例**只有**场景 B 严格版(场景 3)——即"f̂ 丢弃真变量"的情形。**SC 不会因为 spurious / confounder / mediator 违反**——这些是 K20 批评的,但不是 SC 关心的。

**这是 SC 的"窄"特性**:
- ✓ SC 关心:"f̂ 是否反映真变量的重要性"
- ✗ SC 不关心:"f̂ 是否反映了 spurious 关系"
- ✗ SC 不关心:"归因是否符合因果图"

**所以 SC 是 Impossibility 5.1 的"窄"承诺**——它解决了一类问题(预测器丢变量),但不能解决所有问题(spurious / confounder)。

**这意味着**:SC **不**是 K20 批评的完整答案——它**部分**回答。要完整回答需要 Action-ability (T1 示例) + Causal Shapley (H20) + SC 联合。

## 反思(过程记录)

找 5 个反例时,我**开始**以为能找很多 SC 违反的案例。**但跑下来发现 SC 违反其实是"窄"的**——大多数场景 SHAP 归因虽然反映 spurious / confounder, 但**不直接违反 SC**。

**这个发现是反直觉的**——K20 说"Shapley 失败做 importance",但 SC 视角下 Shapley 失败是**有条件的**(只在 f̂ 丢真变量时)。

**我**接下来会想:**SC 的"窄"是好还是坏?**
- **好**:精确,反例清晰,定理严格
- **坏**:不够"完整"——K20 关心的 spurious / confounder 不在 SC 视野里

**这意味着**:完整的"feature importance" axiom 系统可能需要**多个 axiom 联合**——SC + Action-ability + Causal Shapley 各管一摊。这是"扩散性探索"的好例子——**不只 1 个 axiom,而是多个 axiom 联合**。

**反思 200 字**:
> 找 5 个反例时,发现 SC 是个"窄"承诺,大多数场景不违反 SC。这暴露了 SC 与 K20 批评的不同——K20 关心 spurious / confounder,SC 只关心 f̂ 丢真变量。要完整覆盖"feature importance" 需要多个 axiom 联合。这给我一个教训:不要把 1 个 axiom 当成"所有问题的解",扩散性探索意味着多个 axiom 各管一摊。
